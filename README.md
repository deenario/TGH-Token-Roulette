# TGH-Token-Roulette
A roulette created for TGH tokens to be spinned.

## Setup
Download all the node Dependencies using 

```$ npm install```

Run the application with 

```$ node app.js```

Go to ```http://localhost:3000```


## Spin the Wheel
Call the API

```http://localhost:3000/spin``` to spin the wheel and get results.
